  - Medidas de dispersão

  - Amostragem
  - Estimação

- Regressão Linear
  - Regressão Linear simples
  - Regressão Linear Múltipla

- Indicadores de Análise Técnica
  - Rastreadores de Tendência
  - Indicadores de Reversão
